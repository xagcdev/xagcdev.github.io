Manfaat Web Agc

Apa itu web AGC ?. Web AGC adalah singkatan dari Auto Generate Content. Web AGC bekerja dengan cara mengkloning isi web ( baik itu artikel, halaman static, dan lain-lain ) secara otomatis.

Kenapa saya bilang otomatis ?. Karena ketika pemilik website melakukan update artikel, maka artikel yang ada di web agc juga ikut update.

Nah, terus apa manfaatnya buat saya ? Manfaatnya banyak mas.. sumpah beneran kagak bohong..wkwkwk

System Kerja WEB AGC

Cara kerja web AGC yang saya desain ini adalah seperti berikut :

  1. Tentukan Target Domain yang ingin kita AGC.
  2. Masukkan di settingan template ( lihat dokumentasi pemakaiannya di bawah).
  3. Daftarkan web AGC ke GSC (Google Searc Console).
  4. Tunggu sampai semua artikel terindex.
  5. Jika telah mendapatkan trafik yang cukup, lakukan redirect dengan mengaktifkan fitur redirect di settingan templatenya (lihat dokumentas cara mengaktifkan redirectnya di bawah).
  6. Selesai.

Cara Menambah Target Domain Yang ingin Di AGC

Untuk menambah target domain yang ingin di AGC, silahkan ikuti panduan berikut :
1. Buka panel "GENERATE CONTENT" dengan cara menambahkan Kode ini di akhir link :

https://alamat.blogspot.com/?gen

2. Setelah itu, coba kamu kunjungi. maka tampilannya akan menjadi seperti berikut :

3. Silahkan masukkan target domain di kolom input lalu tekan tombol "Generate". Tampilannya akan menjadi seperi berikut :


Perlu kamu ketahui, hasil generate yang dihasilkan ada 2 jenis yaitu :

  1. Output Preview Link yang berfungsi untuk melihat hasil tampilan dari web yang kita AGC.
  2. Output Setting adalah Kode settingan yang akan kita gunakan nantinya.

4. Silahkan copy code "Output SETTING" yang dihasilkan dengan menekan tombol "Copy Setting".

5. Silahkan buat "Postingan Baru" dengan judul terserah kamu lalu isi artikelnya dengan kode "Output SETTING" yang kamu buat sebelumnya. Jika bingung silahkan lihat gambar di bawah.

6. Selesai. Silahkan Lihat Hasilnya.

Tambahan : Pada saat kamu melakukan "Generate Konten", kamu juga bisa menginputkan link artikel konten. Jadi kamu hanya menargetkan AGC untuk 1 artikel doang

Cara Mengaktifkan Fitur Redirect

Perlu kamu ketahui, fitur ini sebaiknya kamu aktifkan ketika web AGC kamu telah terindex di google.

Jika kamu ingin mengaktifkan fitur ini, silahkan ikuti panduan berikut :

  1. Buka dashboard blogger kamu.
  2. Pilih menu "tema" lalu klik tombol "sesuaikan".
  3. Selanjutnya klik "Edit HTML".
  4. Cari kode berikut :
    const redirect = "off";
    const redirectRandom = "no";
    let linkRedirectRandom = ["https://www.bing.com","https://www.youtube.com"];
    let linkRedirectOne = "https://www.google.com";
  5. Lalu ubah kode berikut :
    const redirect = "off";
  6. Ubah menjadi :
    const redirect = "on";
  7. Klik icon "save" yang berada di pojok kanan atas untuk menyimpan.
  8. Selesai.

Catatan : Setelah kamu mengaktifkan fitur redirect url ini, kamu harus menentukan apakah tipe redirectnya random atau hanya satu url. Apa perbedaannya ? silahkan baca settingan selanjutnya dibawah ini.

Cara Redirect Random Url

Fitur ini berfungsi untuk melakukan redirect secara random. Berikut ini settingannya :

  1. Buka dashboard blogger kamu.
  2. Pilih menu "tema" lalu klik tombol "sesuaikan".
  3. Selanjutnya klik "Edit HTML".
  4. Cari kode berikut :
    const redirectRandom = "no";
  5. Ubah menjadi :
    const redirectRandom = "yes";
  6. Lalu cari kode berikut :
    let linkRedirectRandom = ["https://www.bing.com","https://www.youtube.com"];
  7. Silahkan ubah link yang saya beri warna hijau dengan link yang ingin kamu redirect.
  8. Klik icon "save" yang berada di pojok kanan atas untuk menyimpan.
  9. Selesai.

Catatan : Link yang kamu ganti harus mengandung "http://" atau "https://"

Catatan : Link yang kamu ganti harus mengandung "http://" atau "https://"
